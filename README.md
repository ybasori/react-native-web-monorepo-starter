## Installation

Requirement
- npm
- yarn
- [wml](https://github.com/wix/wml)

```bash
yarn install
cd packages/app
wml add ../common ./node_modules/@wow/common
wml start
```
## Run Web

```bash
cd packages/web
yarn start
```

## Run IOS

```bash
cd packages/app
yarn ios
```

## Run And

```bash
cd packages/app
yarn android
```
